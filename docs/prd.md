# 🎮 ShadowMatch

**Core Idea:** Match a shape to its correct shadow from three options
**Core Benefit:** Boosts visual recognition & pattern matching

---

# 1️⃣ Game Vision (User Experience First)

**ShadowMatch** is a **fast-thinking visual puzzle game**.

Player sees a shape →
Chooses the correct shadow from 3 options →
Gets instant feedback →
Moves to the next round.

It should feel:

* Quick
* Clean
* Slightly challenging
* Satisfying when correct
* “I can do better” addictive

---

# 2️⃣ Core Gameplay Experience

## 🔁 Main Loop

1. A shape appears (top or center)
2. Three shadow options appear below
3. Player selects one
4. Instant feedback
5. Next round begins immediately

Game continues:

* With score increasing
* Until mistakes run out OR timer ends

---

# 3️⃣ What the Player Sees

## 🎮 Game Screen

* **Main shape** (clear, solid, colored)
* **3 shadow options** (black/grey silhouettes)
* Score (top)
* Timer or lives (depending on mode)

Everything is centered and easy to scan quickly.

---

# 4️⃣ Decision Feel (Very Important)

Each round should feel:

* “Hmm… quick look…”
* “Got it!” → tap
* Immediate feedback

No delay. No overthinking.
Fast recognition is the goal.

---

# 5️⃣ Feedback System

## ✅ Correct Answer

* Shadow lights up
* Shape briefly aligns or snaps into shadow
* Soft success sound
* Score increases
* Quick transition to next round

---

## ❌ Wrong Answer

* Wrong option shakes or flashes red
* Correct answer briefly highlights
* Small vibration
* Lose life OR time penalty

Player should learn instantly.

---

# 6️⃣ Difficulty Progression

Game becomes harder by:

* Making shapes more similar
* Adding rotation differences
* Adding small detail variations
* Reducing decision time
* Increasing visual complexity

Player should feel:
“That was tricky…”

Not:
“I can’t tell the difference”

---

# 7️⃣ Game Modes (Simple but Effective)

### ✅ Classic Mode

* 3 lives
* Endless rounds
* Difficulty increases

---

### ✅ Time Attack Mode

* 60 seconds
* Score as many correct answers as possible
