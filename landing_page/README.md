# ShadowMatch Landing Page

ShadowMatch is a fast-thinking visual puzzle game where players match shapes to their correct shadows.

## Project Structure

This is a [Next.js](https://nextjs.org) application designed as a professional landing page for the ShadowMatch mobile app.

## Getting Started

1.  **Install dependencies**:
    ```bash
    pnpm install
    ```

2.  **Run the development server**:
    ```bash
    pnpm dev
    ```

3.  **Open in browser**:
    Navigate to [http://localhost:3000](http://localhost:3000) to view the landing page.

## Tech Stack

- **Framework**: Next.js 15+
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Language**: TypeScript
